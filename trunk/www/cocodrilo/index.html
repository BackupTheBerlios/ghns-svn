Cocodrilo: Web interface for GHNS

Cocodrilo is a web interface engine for GHNS. It supports templates and PostgreSQL database installations using the Hotstuff database format. The goal is to support other databases as well, and additionally indirect access via the DXS API.

Functionality will include: listing and sorting entries, searching, rating, adding comments, bookmarking entries... basically everything DXS will offer ;)
