Get Hot New Stuff for Java: jGHNS

The jGHNS library provides access to Get Hot New Stuff resources on remote servers. Beside the data abstraction library, a small download dialog for AWT is available as well. Note that a requirement for the development of jGHNS is that it works on Free Java platforms.

Status: Very premature. It might not even get finished unless someone shows some interest in it. You might want to have a look at the screenshot.

Downloads: jGHNS is licenced under the GNU GPL. You need jikes and kaffe to use it, and the Xerces XML parser library. Also, make (and not ant) is used to build it, although a contributed ant file would be nice to have.
jghns-20050703.tar.gz

Development happens in the Subversion repository, module jghns. See the SVN information page. A mailing list is also available.
